Gmail Setup Guide
  1. Enable 2-Step Verification on your Google account
  2. Go to Google Account → Security → App Passwords
  3. Create an App Password for "SmartRefill"
  4. Copy the 16-character password it gives you
  5. Set these environment variables before running Flask:
set MAIL_USERNAME=you@gmail.com
set MAIL_PASSWORD=xxxx-xxxx-xxxx-xxxx
Your password is never stored in the app — only read from the environment.
